## Instructions to create a new `.env` file

1. In your project root directory, create a new file named `.env`.

2. Open the `.env` file and add the following environment variables:

PORT=3000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=dbpassword
DB_NAME=soul_harmony_pooja_db

3. Save the `.env` file.

> Make sure not to commit your `.env` file to version control if it contains sensitive information.
